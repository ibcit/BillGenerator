-- About BillGenerator --

IBC Bill Generator is a program that allows you to create IBC Airwaybills quickly, and to store the airwaybill information into the Pactrak® system. The data file is loaded into the program from a CSV file. Here are some common attributes for a CSV file:

  • CSV, comma separated values, files are commonly used to transport large amounts of tabular data between either companies or applications that are not directly connected. The files are easily editable using common spreadsheet applications like Microsoft Excel.
  • Fields are separated by commas.
  • Records are separated with system end of line characters, CRLF (ASCII 13 Dec or 0D Hex and ASCII 10 Dec or 0A Hex respectively) for Windows, LF for Unix, and CR for Mac.
  • Fields containing a comma must be escaped with double quotes ( " ).
  • FOR the IBC BillGenerator the csv must contain a header row specifying the data contained within the separator commas.
Essentially, the CSV must conatain the Consignee data (Name, address, city, state, etc...). It may contain also shipment information, such as description, weight and quantity, and other reference values. Each row in the csv file corresponds to one (1) airwaybill.
