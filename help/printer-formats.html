-- Printer Format Options --

Currently, the program supports 3 printer format options:

  • Laser Printer - The Default option which formats airwaybills to print on laser printers.
  • Okidata - Formats output for IBC paper airwaybills on the following models: Okidata 320, Okidata 3410.
  • Lexmark Forms - prints on Lexmark 2408 Forms printers.
Also the program can print on 3 different paper sizes in Laser Printer Mode
  • Letter (8.5" by 11")
    • This is the default setting used for 1 per page airwaybills and 4 per page labels in the US and UK.
  • A5 Paper (5.83" by 8.27")
    • This type of paper is mainly used in the UK for single page airwaybills only.
  • A4 Paper (8.27" by 11.69")
    • This type of paper is mainly used in the UK for single page airwaybills only.
